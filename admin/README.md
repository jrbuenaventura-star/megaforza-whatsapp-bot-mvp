# Megaforza Admin (Next.js)

## Pasos
1. Copia `.env.local.example` a `.env.local` y ajusta `NEXT_PUBLIC_API_BASE` al backend (ej. `http://localhost:3000/api`).
2. Instala dependencias y ejecuta:
   ```bash
   npm install
   npm run dev
   ```
